# 🏔️ Tu Finde en San Juan

**Plataforma digital de alquiler temporario exclusiva para la provincia de San Juan, Argentina.**

Tu Finde en San Juan conecta propietarios y clientes para alquileres temporarios dentro de la provincia, con un fuerte enfoque en la verificación de identidad para garantizar una experiencia segura y libre de estafas.

---

## 🚀 Funcionalidades - FASE 1 (MVP)

- **Verificación Obligatoria:**  
  Validación de identidad (DNI y Selfie) y titularidad de la propiedad por parte de los administradores.

- **Búsqueda y Visualización:**  
  - Vista de lista con filtros por departamento  
  - Vista de mapa con ubicación aproximada para proteger la privacidad  

- **Contacto Directo:**  
  Enlace directo a WhatsApp o llamada para coordinar la reserva.

- **Sistema de Reputación:**  
  Reseñas bidireccionales entre propietarios y clientes.

- **Módulo de Publicidad:**  
  Espacios para banners de comercios locales y Google Ads.

## 🛠️ Stack Tecnológico

### Backend
- **Python** — Lenguaje base del servidor  
- **Django** — Framework web para la lógica de negocio y administración  
- **Django REST Framework (DRF)** — API para conexión con el frontend  
- **Pillow** — Procesamiento de imágenes  
- **PostgreSQL** — Base de datos  

### Frontend
- **React** — Biblioteca de interfaz de usuario  
- **Vite** — Entorno de desarrollo y build  
- **CSS Modules** — Estilos encapsulados por componente  
- **Leaflet & React Leaflet** — Mapas interactivos  

### Herramientas
- **Git & GitHub** — Control de versiones  
- **Postman** — Testing de endpoints

## 🇦🇷 Sobre el Proyecto

Este proyecto nace con la intención de fomentar el turismo interno en la provincia de San Juan, brindando una herramienta segura tanto para el propietario sanjuanino como para el turista que visita nuestros valles.
